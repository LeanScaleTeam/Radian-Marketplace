# call-review-coaching

## When to use

- Weekly curation: this week's best moments → library
- Onboarding: new rep curriculum
- When a rep needs an example of "how to do X"
- Coaching sessions (manager pulls real examples to discuss)

## Core principle

**Top reps' best moves never get studied or taught — they live in individuals.** A call library makes the team's collective wisdom accessible. Audio examples beat written training because tone + pacing + context can't be transcribed.

## Library structure

| Category | Sub-categories |
|---|---|
| **Great discovery** | Opening, pain identification, quantification, multi-thread, next-step setting |
| **Objection handling** | Price, competitor, timing, budget, no-decision |
| **Demo / pitch moments** | Story-telling, ROI framing, vision selling |
| **Closing / negotiation** | Price holding, terms, urgency creation |
| **Hard moments** | Hostile prospects, mid-deal slips, recovery from misstep |
| **Anti-patterns** | What NOT to do (with anonymization) |

## Workflow

1. Continuous scoring of every call (from `sales-coaching-at-scale`)
2. Surface top-tier moments (top X% on specific behaviors)
3. Trim to 1-3 min clips with context
4. Tag with category + sub-category + skill demonstrated
5. Add coaching commentary (why this is great, what to notice)
6. Add to library searchable by category + by rep + by deal type
7. Push curated weekly digest to team

## Per moment captured

```markdown
*Library Entry — [Title]*

*Category:* Great Discovery / Pain Quantification
*Rep:* [name]
*Account:* [anonymized or named with permission]
*Date:* [date]
*Duration:* 2:18

*What happens:*
[1-2 sentence summary]

*Why it's great:*
- [Specific skill 1]
- [Specific skill 2]
- [Specific skill 3]

*What to listen for:*
- 0:14 — opens with surprising question
- 0:42 — moves from adjective pain to dollar quantification
- 1:30 — multi-thread pivot — "who else is feeling this?"

*Coachable takeaway:*
"When the prospect says 'we're losing deals to competition', go straight to quantifying — 'how many deals last quarter? what's average ACV?' The pain isn't real until they've quantified it themselves."

*Audio clip:* [link]
*Full call link (with context):* [link]
```

Weekly digest:

```markdown
*Call Library — week of [date]*

*New additions this week:*
1. [Title] — Great discovery — [Rep]
2. [Title] — Objection handling — [Rep]
3. [Title] — Anti-pattern — anonymized

*Most-listened-to entries this week:*
- ...

*Suggested coaching plays (1:1s next week):*
- For new reps: start with "How [rep] opens a disco call"
- For struggling on price: review "Hold price under pressure" example
```

## Connectors

- Gong / Fireflies / Chorus (required)
- Slack (delivery)
- LMS / training platform (if used)

## Gotchas

- Get rep + customer consent for sharing (especially externally)
- Anonymize anti-patterns — never name and shame
- Keep clips short (1-3 min) — full calls are too much for casual review
- Refresh quarterly — examples decay as messaging / product / methodology evolve

# sales-coaching-at-scale

## When to use

- Continuous (after every recorded call)
- Weekly manager 1:1 prep
- Onboarding new reps (faster ramp via coaching density)
- When ramp time / quota attainment correlates with coaching frequency

## Core principle

**Sales coaching is the #1 driver of rep quota attainment** (CSO Insights). A 10-rep team running 5 calls/week = 50 calls — no manager can review them all. Top reps and bottom reps differ on a measurable handful of call behaviors.

## Behaviors to track per call

| Behavior | Source |
|---|---|
| Talk ratio (rep vs. customer) | Transcript |
| Question count | Transcript |
| Question depth (follow-ups, "why" chains) | Transcript |
| Pain quantification attempts | Transcript |
| Objection handling style | Transcript |
| Discovery dimensions hit (methodology coverage) | Transcript |
| Filler words / pace | Transcript |
| Next-step clarity | Transcript |
| Stakeholder mentions | Transcript |

## Workflow

1. Continuously analyze each recorded call against the behavior set
2. Per call: produce a rep-facing coaching note (light, 1-2 focuses)
3. Per rep: maintain a coaching trend dashboard
4. Per manager: weekly synthesis highlighting which calls to review
5. Manager spot-reviews 2-5 flagged calls per rep per week (high-leverage)
6. Track behavior change over time — does coaching produce improvement?

## Per-call output (rep-facing)

```markdown
*Coaching Note — [Rep] — Call: [Account] — [Date]*

*Quick view:*
- Talk ratio: 38% (target 30-40%) ✅
- Questions asked: 9 (target 11+) 🟡
- Why follow-ups: 2 (target 3+)
- Discovery dimensions hit: 4 of 6

*One focus for next call:*
**Go one layer deeper on pain quantification.**

[Specific example with timestamp + quote]

*One thing you did well:*
[Specific moment, timestamp, what worked]
```

## Per-rep weekly trend

```markdown
*Coaching Trend — [Rep] — Last 4 weeks*

| Behavior | This wk | Last wk | 4-wk avg | Trend |
| Talk ratio | 38% | 42% | 41% | improving ⬆️ |
| Questions | 9.2 | 7.8 | 8.1 | improving ⬆️ |
| Why follow-ups | 2.1 | 1.4 | 1.6 | improving ⬆️ |

*Coaching focus this 4 weeks:* "Go deeper on pain quantification"
*Result:* Why-follow-up count up 50%

*Suggested next focus:* Discovery dimension coverage (still missing 1-2 per call)
```

## Connectors

- Gong / Fireflies / Chorus (required)
- CRM (deal context)
- Slack (delivery to reps + managers)

## Gotchas

- AI coaching < manager coaching for the highest-stakes moments — combine, don't replace
- Don't dump 10 critiques per call — ONE focus per cycle moves behavior
- Track behavior change AS WELL as quota change — both signals matter
- Sensitive moments (HR, sensitive customer convos) — don't analyze, flag and skip
